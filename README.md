# Crop recommendation system

### Tech stack
1. Frontend - ReactJs
2. Api Integration - NodeJs
3. Backend - Java (the backend is in different repository)
4. Predcition model - Python (the backend is in different repository)
5. Deployment - Docker, Kubernetes
